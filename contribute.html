Contribute to Thamizha projects...

You can get involved in Thamizha project in various ways.
Some of the ways you can help are...

- If you are a software programmer, you can help in developing or fixing the bugs in the existing thamizha projects.
Check our github page https://github.com/thamizha
- If you have used any of the thamizha softwares and liked them,you can write about thamizha projects in your blog, facebook or twitter.

- If you have used thamizha projects and found any problem, then report the issues in thamizha projects in our issues page.

- Also you can help to test the thamizha projects and help to identify the issues in them.

- You can also help by teaching your friends and family about using tamil in computers.
